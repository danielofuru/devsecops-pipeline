# DevSecOps Pipeline on AWS

A production-grade DevSecOps pipeline built from scratch on AWS.

## What This Project Covers
- Infrastructure as Code with Terraform (VPC, ECS, IAM, Security Groups)
- CI/CD pipeline using GitHub Actions
- Container security scanning with Trivy
- IaC security scanning with tfsec and Checkov
- Automated deployment to AWS ECS
- Monitoring and alerting with CloudWatch and Grafana
- Blue/Green deployments with AWS CodeDeploy
- Audit logging with CloudTrail and AWS Config
